## Requisitos
- **Python 3.10+**
- **pip** (instalado com o Python)
- (Opcional) **venv** para ambiente virtual

## Passos rápidos
```bash
git clone https://github.com/Deni-jpg/Loja-Checkpoint.git
cd Loja-Checkpoint

# (Opcional) criar e ativar venv
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS/Linux
source .venv/bin/activatesource .venv/bin/activate

# Instalar dependências
pip install -r requirements.txt

# Executar
